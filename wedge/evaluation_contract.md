# Evaluation Contract

The wedge is successful only if raw artifacts show all of the following:

- the frozen core remains unchanged
- the accepted abstraction is verifier-backed
- the abstraction reduces contradiction on held-out tasks
- positive transfer succeeds when role remapping is structurally valid
- negative transfer is blocked for principled reasons
- the live model proposal path does not mutate ontology directly
- the strongest live model baseline does not match DENSN on contradiction relief plus verifier-backed transfer quality
